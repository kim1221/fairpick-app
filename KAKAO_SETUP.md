# Kakao REST API 키 발급 방법

## 1. Kakao Developers 가입 및 앱 생성

1. **Kakao Developers 접속**
   - https://developers.kakao.com/ 접속
   - 카카오 계정으로 로그인

2. **애플리케이션 추가**
   - 우측 상단 "내 애플리케이션" 클릭
   - "애플리케이션 추가하기" 클릭
   - 앱 이름: `Fairpick` (원하는 이름)
   - 사업자명: 개인 또는 회사명 입력
   - "저장" 클릭

## 2. REST API 키 확인

1. **앱 설정 페이지 이동**
   - 생성한 애플리케이션 클릭
   - 좌측 메뉴에서 "앱 키" 클릭

2. **REST API 키 복사**
   - "REST API 키" 항목의 키 값 복사
   - 예시: `1234567890abcdef1234567890abcdef`

## 3. 플랫폼 등록 (선택사항)

1. **플랫폼 설정**
   - 좌측 메뉴에서 "플랫폼" 클릭
   - "Android 플랫폼 등록" 또는 "iOS 플랫폼 등록" 클릭
   - 패키지명/번들 ID 입력

## 4. 프로젝트에 키 설정

### 방법 1: 환경 변수 파일 사용 (추천)

```bash
# 프로젝트 루트에 .env 파일 생성
echo "KAKAO_REST_API_KEY=발급받은_REST_API_키" > .env
```

### 방법 2: 직접 코드에 입력

`src/utils/geocoding.ts` 파일 수정:

```typescript
const KAKAO_REST_API_KEY = '발급받은_REST_API_키';
```

## 5. API 사용량 확인

- Kakao Local API (역지오코딩)
- **무료 할당량**: 300,000건/일
- **충분한 사용량**: 일반적인 앱에서는 무료 할당량으로 충분

## 6. 주의사항

⚠️ **보안 주의**
- REST API 키는 클라이언트에 노출되므로 보안에 주의
- 프로덕션 환경에서는 백엔드에서 역지오코딩 처리 권장
- `.env` 파일은 `.gitignore`에 추가하여 Git에 커밋하지 않기

## 7. 테스트

앱을 실행하고 위치 권한을 허용하면:
- 헤더에 "성수2가1동" 같은 행정동 이름이 표시됩니다
- 모든 이벤트 카드에 거리와 이동 시간이 표시됩니다

---

**문제 발생 시:**
- Kakao Developers 콘솔에서 API 호출 통계 확인
- 브라우저 개발자 도구에서 네트워크 요청 확인
- 로그에서 `[Geocoding]` 태그로 에러 확인
